COMMON ISSUES & FIXES
=====================

Issue: Socket.IO not connecting (red dot)
Fix: 
  - Check backend is running on port 8001
  - Check browser console for errors
  - Restart both backend and frontend

Issue: No real-time updates
Fix:
  - Check green dot is showing (Socket.IO connected)
  - Check browser console for Socket.IO events
  - Check backend terminal for emit logs
  - Run test script to verify

Issue: Today's calls showing 0
Fix:
  - Make a test call
  - Check browser console for errors
  - Verify backend is returning calls
  - Check: http://localhost:8001/calls

Issue: Times showing wrong timezone
Fix:
  - Clear browser cache
  - Hard refresh (Ctrl+Shift+R)
  - Check dateUtils.js is being used
